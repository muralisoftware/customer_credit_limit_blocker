WhatsApp Integration for Odoo 17

This module allows users to send WhatsApp messages directly from Odoo forms like Sales Orders, Invoices, and Contacts.

Key Features

  • Send WhatsApp messages from partner, invoice, or order forms
  • Uses WhatsApp Web (https://wa.me)
  • Dynamic message template support

How It Works

  1. Install the module
  2. Click "Send WhatsApp" on any record
  3. Preview message and send
